## Diagrama de Fluxo de Dados (Pub/Sub)

```mermaid
sequenceDiagram
    participant App as Rust App
    participant API as cyclonedds API
    participant FFI as cyclonedds-rust-sys
    participant C as CycloneDDS C
    participant Net as DDS Network

    Note over App: Define topic type<br/>#[derive(DdsType)]
    
    App->>API: DomainParticipant::new(0)
    API->>FFI: dds_create_participant()
    FFI->>C: dds_create_participant()
    C->>Net: Discovery (SPDP)
    Net-->>C: Participants discovered
    C-->>FFI: participant handle
    FFI-->>API: DomainParticipant
    API-->>App: participant

    App->>API: Topic::<T>::new(...)
    API->>FFI: dds_create_topic()
    FFI->>C: register type + create topic
    C-->>FFI: topic handle
    FFI-->>API: Topic<T>
    API-->>App: topic

    alt Publisher
        App->>API: DataWriter::new(...)
        API->>FFI: dds_create_writer()
        FFI->>C: create writer
        C-->>FFI: writer handle
        FFI-->>API: DataWriter
        API-->>App: writer
        
        App->>API: writer.write(&msg)
        API->>FFI: dds_write()
        FFI->>C: serialize (CDR)
        C->>Net: UDP multicast / unicast
    end

    alt Subscriber
        App->>API: DataReader::new(...)
        API->>FFI: dds_create_reader()
        FFI->>C: create reader
        C-->>FFI: reader handle
        FFI-->>API: DataReader
        API-->>App: reader
        
        Note over Net: Data arrives
        Net->>C: UDP packet
        C->>C: deserialize (CDR)
        
        App->>API: reader.take()
        API->>FFI: dds_take()
        FFI->>C: read from RHC
        C-->>FFI: samples + info
        FFI-->>API: Vec<T>
        API-->>App: samples
    end

    alt Async Stream
        App->>API: reader.read_aiter()
        API->>API: create WaitSet + attach
        loop While stream active
            API->>FFI: wait_async()
            FFI->>C: dds_waitset_wait()
            C-->>FFI: triggered
            FFI-->>API: Vec<T>
            API-->>App: batch.await
        end
    end
```
